NITCbase

NITCbase is an online Database Management tutoring system that is intended to help undergraduate students understand the concepts of DBMS through the creation and manipulation of relations and records. This project presents a step-by-step guide for the students to implement the data structures and process the query that can manipulate the data stored in the disk using the XFS interface provided by NITCbase. Students are provided with learning modules which enhances their knowledge of the concepts as well as the system.


There are seven layers to be implemented in this project. Starting from the bottom layer, the Physical layer to the top layers Algebra and Schema layer. NITCbase provides a front-end command-line interface for query processing. The implementation is to be done in C++ throughout the project.
