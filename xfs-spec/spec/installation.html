XFS Interface Installation Guidelines

XFS Interface is a command line external interface to access the NITCBase filesystem from the host (UNIX) system. The filesystem is simulated on a binary file. This interface aims to provide the functionality of transferring relations from/to the host system to/from the disk and can be helpful in debugging the system during implementation. .There are three types of commands that can be given as input to the XFS interface:

  1. External File System Commands
  2. Data Definition Language Commands
  3. Data Manipulation Language Commands

Setting Up

The following are the instructions to be followed for installation in linux:

  1. Download the NITCBase XFS Interface package from here.

  2. Copy the tar file to your home directory.
    cp NITCBase.tar.xz $HOME/
    cd $HOME

  3. Extract the contents using the command.
    tar -xvf NITCBase.tar.xz
    Now you will have a directory NITCBase in your home directory, with components required for XFS Interface.

  4. Install prerequisites such as g++
    For Debian-based linux distros:
    sudo apt install g++

  5. Change directory to xfs-interface directory.
    cd $HOME/NITCBase/xfs-interface

  6. Make to build all the components:
    make

Running XFS Interface


  • To run the xfs interface:
    ./xfs-interface
