System Design

NITCbase is a tiny database management system. It has a very simple specification that allows a junior undergraduate computer science student to implement it in a few months.

Design Buffer Layer Cache Layer B+ tree Layer Block Access Layer Algrbra Layer Schema Layer Physical Layer Constants
Frontend Interface

NITCbase provides a command line interface to the users. A user can execute any query using the provided command line interface.

Front End Interface Specification
Support Tools

NITCbase is provided with some support tools to help implementing it. These tools can be helpful in debugging the system during implementation.

XFS Interface

XFS Interface is a command line external interface to access the NITCBase filesystem from the host (UNIX) system.
