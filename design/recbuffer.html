getRecord

Arguments:

record (union Attribute *), slot_num (int)
where record is a pointer to an array of type Union Attribute whose size is assumed to be equal to the number of attributes in the relation.

Return value:

SUCCESS On succesful copy of the record.
E_OUTOFBOUND If the slot_num is not in the range of number of slots per the block.
E_FREESLOT If the slot corresponding to the slot_num is free.
