Block Access Layer

In any database management system, in order to retrieve any data from the database or to alter the schema of the relations in the database the system has to work with the disk blocks. Block Access layer provides the abstraction and hides the disk structures to the above layers i.e, algebra layer and schema layer and also provides interface to the above layers in terms of records. Hence, the Block Access layer processes the requests from the algebra and schema layers and works with disk blocks.
